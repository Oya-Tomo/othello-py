## ファイル構成

- game.py : オセロのゲーム
- othello.py : オセロのロジックを含むクラス
- model.py : モデルの構築
- train.py : モデルの学習
- make_data.py : 学習用データの作成＆ファイルへ圧縮
- extract_data.py : CSV ファイルからデータを取り出す
- ggf_split.py : ggf 形式からの切り出し（現在使用していない）

## 基本説明

この AI は畳み込み学習を利用して、オセロの盤面の特徴を学習しプロの棋譜をもとにした手を打つことの出来るプログラムです。

## 動かし方

### セットアップ

```
python3 -m venv lib
pip install -r requirements.txt
```

### モデルの学習

```
./lib/bin/python make_data.py
./lib/bin/python model.py
./lib/bin/python train.py
```

### ゲームの実行

```
./lib/bin/python game.py
```

## 動作

- モデルの作成＆学習。
  - CSV の読み込み
  - CSV のデータ → 盤面データ
  - \*.npz 形式で保存（バカでかい → データの削減）
- オセロのアプリに組み込んで動作させる。
  - オセロモデルの学習に使ったコアクラス（オセロのルールロジックが書いてある）を流用
